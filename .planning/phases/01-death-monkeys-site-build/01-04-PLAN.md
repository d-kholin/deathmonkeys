| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-04-01 | Tampering | XSS via card fields (title/description/summary/callsign) | mitigate | Card props rendered through Astro `{expr}` interpolation (auto HTML-escaped); no `set:html` on collection content. Data is first-party seed, already zod-validated in Plan 1.2. |
| T-04-02 | Information disclosure | External placeholder image host | accept | `picsum.photos`/`placeholder.com` over HTTPS; no PII in request; out-of-scope to self-host per REQUIREMENTS.md. |
| T-04-03 | Spoofing | Broken `/gallery/${id}` links if id mismatches | mitigate | `entry.id` passed directly from the same collection that `getStaticPaths` enumerates in Plan 1.5, guaranteeing link/route parity. Verified by build (no 404s in static output). |
