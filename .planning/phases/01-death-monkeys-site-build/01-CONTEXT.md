<decisions>
## Implementation Decisions

### Hero Section (Home Page)

- **D-01:** Hero height is 60–70vh — substantial presence without forcing a full-screen barrier before content.
- **D-02:** Hero background uses a subtle CSS noise/grain texture overlay (SVG data URI) over the near-black `#0a0a0a` base — no extra assets, pure CSS.
- **D-03:** Hero tagline: **"Armed. Dangerous. Slightly Bananas."** — leans into the Death Monkeys brand humor, not geography-specific.

### Typography

- **D-04:** Monospace headings use **JetBrains Mono** loaded via Google Fonts. Applied to all headings; uppercase treatment for `DEATH MONKEYS` wordmark and major section headings.
- **D-05:** Body text uses the system sans-serif stack (`ui-sans-serif, system-ui, sans-serif`) — no additional font requests.

### Mobile Navigation

- **D-06:** Hamburger opens a **slide-down dropdown panel** below the nav bar (not full-screen overlay). Links stack vertically in the panel.
- **D-07:** Tapping any nav link **auto-closes** the mobile menu — one-tap navigation, handled in the vanilla JS `<script is:inline>`.

### Seed Data

- **D-08:** Roster first entry: callsign **Nakunga**, display name **Naku**. Add 2 invented teammates with plausible airsoft handles (e.g., Gravedigger, Jackdaw).
- **D-09:** Event and gallery seed content uses a **committed bit** — real-feeling Death Monkeys mission names and flavor text (e.g., "Op: Primate Protocol", "Black Banana Engagement"). Not Badlands/Alberta-centric; lean into the team's brand identity.
- **D-10:** Roles and loadout descriptions for roster entries should be specific and airsoft-realistic (e.g., "Assault / HPA rifle", "Support gunner / LMG platform").

### Claude's Discretion

- Exact invented teammate callsigns and roles (user gave anchor: Nakunga/Naku; rest is open)
- Specific event dates, locations (keep generic/fictional — no real Badlands references required)
- Gallery AAR photo counts and placeholder image service (`placeholder.com` or `picsum.photos`)
- About page copy (placeholder text that fits the tactical-silly tone)
- Exact nav bar height and spacing
- Status badge exact Tailwind classes (follow `#6b7c4a` olive for upcoming, gray for completed, muted red for cancelled as specified in REQUIREMENTS.md)

</decisions>

<specifics>
## Specific Ideas

- **Hero tagline locked:** "Armed. Dangerous. Slightly Bananas." — tone is tactical-silly, not serious military.
- **Nakunga / Naku** is the real team member in the roster seed. Other entries are invented.
- **JetBrains Mono** for headings — loaded via Google Fonts CSS `@import` in the global stylesheet.
- **Noise texture:** implement as a CSS `background-image` using an SVG data URI `<svg>` with `feTurbulence` filter, low opacity overlay on the hero element.
- **Death Monkeys mission naming convention:** "Op: [Primate-adjacent or absurdist name]" — gives the site personality without requiring real event history.

</specifics>
